Se pueden crear errores con console.error("este aparece solo en consola");

Tipos de datos:

  • String: Es una cadena de caracteres, se puede escribir con comillas dobles("") o simples('')
  • Numeros: Nada que decir, son numeros...
  • Booleanos o logicos: Son estados que se componen en true o false lo que seria equivalente a 1 y 0 respectivamente
  • Array: se usan para hacer listados, la forma de usarlo es asi:
    var array = ["alvaro", true, false, 3.5, "etc"]
    si se desea acceder a el se puede hacer de la siguiente forma:
    console.log(array[1]); (esto traaera la posicion 1 del array osea true)
  • Objeto: Es una representacion abstracta de un objeto de la vida real, los objetos estan conformados por pares de clave y valor, clave seria un nombre clave para reconocer esa propiedad y valor seria el dato que se le asigna a la clave. Un ejemplo de objeto seria una persona que a su vez tiene atributos como podrian ser peso, altura, color de ojos, edad, etc 
    •             Dos ejemplo de objeto serian los siguientes:

            Ejemplo 1:
                var persona = {
                    nombre: "alvaro",
                    apellidos: ["Toro", "Martínez"],
                    obesidad: true,
                    altura: 1.75,
                    edad: 25,
                }

                    Se declara la variable y se le asigna con el signo igual(=) a el objeto que esta contenido entre llaves({}), luego dentro de las llaves se pondran las
                    propiedades, poniendo primero el nombre de la propiedad(clave), luego dos puntos(:) seguido de la variable(valor) que vamos a asignar a dicha propiedad 
                    y terminamos con una coma para poner la siguiente propiedad (Si no asignamos una variable a la propiedad, esta quedara cono undefined, NO como null)


            Ejemplo 2:
                var persona = new Object();
                persona.nombre = "alvaro";
                persona.apellidos = ["Toro", "Martínez"];
                persona.obesidad = false;
                persona.altura = 1.75;
                persona.edad = 25;

                    primero se declara la variable y esta se asigna a un nuevo objeto, despues llamamos el objeto con el nombre de la variable(persona) seguido de un punto
                    y el nombre de la propiedad que vamos a asignar (nombre), luego ponemos el signo igual(=) para asignarle la variable (alvaro) sea string,
                    boolean, array, etc y debemos finalizar con punto y coma(;)

SWITCH-CASE:

      Se conforma de 3 o 4 palabras:
        1-switch(){}:dentro de los parentesis va la variable que se quiere analizar y dentro de las llaves los casos
        2-case: se pone la palabra case seguido de las posibles asignaciones de la variable asignada
        3-break: es vital usar el brake al finalisar cada caso
        4-default: se usa como una especie de else
      
      Ejemplo:

        let typeCard = "credit card";

        switch(typeCard){
          case "creditCard":
            console.log("esta es una tarjeta de credito");
            break;
          case "debitCard":
            console.log("esta es una tarjeta de debito");
            break;
          default:
          console.log("no tienes tarjeta de credito");
        }

BUCLES:

Ciclo while:

      let count = 50;

      while(count > 0){
        console.log("hola jeje");
        count--;
      }

        nota:en este ejemplo aparecera el "hola jeje" y el conteo disminuira uno a la vez en cada ciclo
        ALERTA: Si no se pone un limite a la condicion, se creará un bucle infinito

Ciclo for:

      let names = ["ryan", "joe", "jhon"];

      for(let i = 0; i < names.length; i++){
        console.log(names[i]);
      }

      otro ejemplo sería:
      for(let i = names.length - 1; i >= 0; i--){
        console.log(names[i]);
      }

FUNCIONES:

      Primero se pone la palabra clave "function" seguiodo del nombre de la funcion, luego parentesis (dentro de estos van los parametros como por ejemplo nombre).
      Luego al momento de llamar la funcion se deben ingresar los parametros para que la funcion se ejecute bien

      function saludar(nombre, apellido){
        let nombre;
        console.log("hola " + nombre + " "+ apellido + ", ¿como estas?");
      }

      SE DEBE LLAMAR LA FUNCION!
      saludar("alvaro", "toro martinez");

      el resultado seria:
        hola alvaro toro martinez, ¿como estas?

      EJEMPLO 2:

        function add(n1, n2){
          console.log(n1 + n2);
        }

        add(800, 1000);

        El resultado sería 1800


        document.write(); es una funcion (todo lo que va despues de un punto es una funcion, en este caso write) y dentro de los parentesis se recibe el parametro
