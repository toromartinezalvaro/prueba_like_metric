Nuevas formas de declarar variables

  • var: Con var podemos redeclarar variables de la siguiente forma
            var nombre = 'juan';
            var nombre = 'pedro';
            var nombre = 'andres';
            
  • Let: No se puede asignar varias veces (no se puede poner let varias veces para la misma variable)
            let nombre = 'juan';
            let nombre = 'pedro';           //aqui sale error por que estoy usando let de nuevo para la misma variable
            let nombre = 'andres';          //aqui sale error por que estoy usando let de nuevo para la misma variable

            Ademas de lo anterior, let tiene scope de tipo bloque (si se declara dentro de una funcion no se puede usar fuera de la misma)
  • const: al igual que let, no se puede declarar varias veces. Ademas de lo anterior, cons tiene scope de tipo bloque (si se declara dentro de una funcion no se puede usar fuera de la misma). Si usamos const para un arreglo [] podemos usar el metodo .push() para agregarle datos (solo en arreglos podemos modificar una constante)
