        this.nombre = nombre,
        this.edad = edad,
        this.correo = correo
    }

    //nuevo metodo que recibe y retorna los parametros del constructor de arriba
    mostrarInfo(){
        return `

HERENCIA

        Se usa cuando se quieren heredar todas las funciones y parametros de una clase desde otra clase, la forma de hacerlo es la siguiente:

        class clase1 {                          //esta es la clase madre

        }

        class heredando extends clase1 {            //esta es la clase que hereda, se pone la clase(heredando) luego "extends" y luego la clase de la que hereda

        }

            En el anterior ejemplo la clase "heredando" va a obtener todos los parametros y metodos de la clase madre llamada "clase1"

        Ademas de heredar todas las propiedades y funcionalidades de la clase que extiende, tambien puede tener nuevos parametros o metodos, para agregarlos hariamos lo
        siguiente (agregar el metodo super(con los parametros de la clase madre aqui); seguido del constructor(){})
