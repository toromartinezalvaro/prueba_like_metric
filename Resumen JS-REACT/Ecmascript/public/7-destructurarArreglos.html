Destructurar arreglos

    Cuando se necesita trabajar con un arreglo de muchos elementos, contar la posicion no es una opcion, por lo cual podriamos destructurar el arreglo (transformar los datos a variables para acceder mas facilmente a ellos)

        Ejemplo

        const persona = ['alvaro', 23, 'colombia', 'desarrollador web'];

        const [nombre, edad, pais, profesion] = persona;            <------ con esta linea estamos asignando a cada posicion un nombre de variable que podemos llamar posteriormente 

        console.log(pais);

        si quisieramos saltarnos la asignacion de algun parametro solo pondriamos una coma con un espacio asi:

        const [nombre, edad, , profesion] = persona;                Aqui nos saltariamos el pais
